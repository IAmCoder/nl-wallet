# App Store

This folder contains the metadata used to publish the app to the Apple App Store.
The App Store has the following fields, the character limit is provided when applicable.

- Name `(30)`
- Subtitle `(30)`
- Promotional Text `(170)`
- Description `(4000)`
- Keywords `(100)`
- Privacy Policy URL
- Support URL
- Marketing URL
- Copyright
